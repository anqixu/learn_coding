# learn_coding

Interactive coding exercise runners modeled after [Rustlings](https://github.com/rust-lang/rustlings). Fix broken or incomplete code to advance.

## Projects

| Project | Language | Exercises | Status |
|---------|----------|-----------|--------|
| [cpplings](cpplings/) | C++ (11 through 23) | 280 | ~42% high-quality, rest placeholders |
| [pytorchlings](pytorchlings/) | Python / PyTorch | 85 | 100% implemented |

See each project's `README.md` for build/run instructions and a full TODO list.
